## 布隆过滤器
特点：
高效地插入和查询，占用空间少，返回的结果是不确定性的.
一个元素如果判断结果为存在的时候元素不一定存在；但是判断结果为不存在的时候则一定不存在
布隆过滤器可以添加元素，但是不能删除元素；因为删掉元素会导致误判率增加。
误判只会发生在过滤器没有添加过的元素；对于添加过的元素不会发生误判。已经加进来的不会误判。(注意)

误判性的原因：由于外面某各元素算出的hash值，刚好跟里面已有的hash值一样（hash冲突）

一个超大的集合中，判断某个元素是否存在
有，是极大可能有
无，是一定无
